# keenetic_vpn_bot

Работа через Edge, следовательно - Винда

Установка с венв:
```
py -m venv venv
```
```
.\venv\Scripts\Activate.bat
```
```
pip install -r requirements.txt
```
```
py telegrambot.py
```
После можно создать батник:

```
cd .\*** (путь до папки)
.\venv\Scripts\Activate.bat
py telegrambot.py
```
